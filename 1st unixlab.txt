       codes learnt in unix programming and shell.
    1. x - delete character
    2. u - undo
    3. ctrl-R - redo
    4. A - to append text at the end
    5. :wq - to save and exit
    6. :w - save
    7. :q! - trash changes
    8. dw - moves cursor to beggining and delete
    9. 2w -  to move the cursor 2 words forward
    10. 3e - moves the cursor end of third word forward
    11. 0 - move to the start of the line
    12. d2w - deletes 2 words
    13. 2dd - deletes 2nd line
    14. p - puts the previously deleted text after the cursor
    15. re - to replace the letter with e
    16. ce - deleted the word and places you in insert mode
    17. G - to move to the bottom of the file
    18. gg - to move to the start of the file
    19. % - to find matching ),],}
    20. :s/old/new where old gets replaced by new
    21. / = backward search
    22. ? = forward search

    29. yw - yanks one word
    30. o - ctrl-w jumps fron 1 window to another
